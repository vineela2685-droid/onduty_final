🚀 Setup Instructions

1

Install Dependencies

Run npm install to install Express, Mongoose, and other backend dependencies

2

Update Database Password

Copy server/.env.example to server/.env and set MONGODB_URI to your connection string (preferred):

MONGODB_URI=mongodb+srv://<username>:<password>@<cluster>.mongodb.net/onduty?retryWrites=true&w=majority
3

Start Backend Server

Run npm run server:dev to start the Express server with MongoDB connection

4

Start Frontend

In a new terminal, run npm run dev to start the React development server

5

Test Connection

Log in as an instructor/admin and click "MongoDB Test" to verify the connection

🔧 Troubleshooting

  • Connection Error: Check your MongoDB connection string and password
  • Port Conflict: Ensure port 5000 is available for the backend server
  • CORS Issues: Backend is configured to allow requests from the frontend
  • Dependencies: Make sure all npm packages are properly installed
  • Network: Verify MongoDB Atlas network access and IP whitelist
